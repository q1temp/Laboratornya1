## Задача 1



### Текст задачи



Сформировать список из названий функций модуля List



### Алгоритм решения



1) Попадаем в функцию main

2) Вызываем функцию printListMethods

3) В функции printListMethods получаем тип модуля ListModule из сборки F# через рефлексию

4) Проверяем есть ли модуль List, если модуль list не найден (t = null), то выводим ошибку "Модуль List не найден". Если модуль найден, то получаем все методы модуля через GetMethods (), создаем список имен методов, List.distinct - удаляем дубликаты, List.sort - сортируем имена по алфавиту, List.iter - выводим каждое имя на новой строке

6) Завершаем программу

# Задание 2



## Задача 1



### Текст задачи



Возвратить true, если число состоит только из чётных цифр, и false — в противном случае



### Алгоритм решения



1) Попадаем в функцию main

2) Запрашиваем число у пользователя и считываем его

3) Преобразуем строку в число и пишем abs для отрицательных чисел

4) Вызываем рекурсивную функцию proverkachetchicel с числом

5) Если число <10 (одна цифра), то возвращаем true, если цифра четная (then number % 2 = 0), иначе false

6) Если число >10 (2+ цирф), то берем посл.цифру (lastchiclo = number % 10). Если последняя цифра нечетная возвращаем false. Если последняя цифра четная, то рекурсивно вызываем функцию для number / 10 (отбрасываем последнюю цифру), и если попадается нечетная цифра возвращаем  false, если все цифры четные то возвращаем true.

7) Результат, который получился сохраняем в result (true or false)

8) Выводим "Число %d состоит только из четных чисел: %b"
